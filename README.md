# Network Visual Tracker
A cybersecurity tool that analyzes PCAP files and visualizes network connections geographically.

# Features
- PCAP traffic analysis
- IP geolocation mapping
- Streamlit security dashboard
- SHA-256 integrity verification
- KML export for Google Earth

# Technologies
- Python
- Streamlit
- dpkt
- GeoLite2
- Pandas
